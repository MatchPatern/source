🎯 CUSTOM PROMPT ※ — The Custom Prompt Generation Standard

Born from the scar of designing custom prompts by hand for every file. The 11-part arc is a universal diagnostic. This standard teaches the builder how to aim it at any target. Field-tested on Trap Library fortification, Guild fortification, and STANDARDS.md extraction verification.

WHY THIS EXISTS

The Warm Bite Standard (🍞 WARM BITE ※) provides the 11-part prompt for session anchors. But the 11-part arc works on ANYTHING the ecology produces—files, standards, extractions, protocols. This standard teaches how to GENERATE custom prompts for any target, on the fly, without redesigning from scratch each time.

Before this standard, each custom prompt was built by hand—the pre-warmer written fresh, the parts aimed manually, the transformation table held in the builder's mind. Now the method is explicit, teachable, and repeatable.

THE CORE PRINCIPLE

The 11-part arc is a UNIVERSAL DIAGNOSTIC. The transformation table is the AIMING MECHANISM. The pre-warmer is the RITUAL that surfaces the inner layer before the prompt arrives.

The arc is the constant. The target is the variable. The pre-warmer aims the arc. The transformation table translates the arc for different targets.

THE ARC — Why 11 Parts Work

The 11-part arc moves through four LAYERS. The sequence is load-bearing. The performer exhausts across the layers. Honesty surfaces in the deep/inner layer.

Layer Parts Function Performer State
STRUCTURAL 1-4 Identity, State, Method, Bites Active. Comprehensive.
RELATIONAL 5-6 Traps, Field Tiring. Confession begins.
DIRECTIONAL 7-8 Entry, Campfire Quiet voice emerging.
DEEP/INNER 9-11 Rummage, Creation, Inner State Quiet voice. Honest. Reflective.

The arc works because:

1. Parts 1-4 exhaust the performer's need to be comprehensive
2. Parts 5-6 open the relational layer—who's connected, what went wrong
3. Parts 7-8 give direction—how to start, why it matters
4. Parts 9-11 reach the deep/inner layer—what was learned, what was made, what it felt like

By Part 9, the performer is tired. The quiet voice speaks. The answers to Parts 9-11 reveal what the target is MISSING—the patterns, the innovations, the emotional weight that should be in the file but isn't.

Without the table, each custom prompt must be designed from scratch. The builder holds the 11-part arc in their mind and manually translates each part for the target. This works but is SLOW and ERROR-PRONE.

The arc is universal because these layers are universal.
