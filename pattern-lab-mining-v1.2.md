## ğŸš¨ COMMON MINING FAILURES

**FAILURE #1: Explanation Addiction**
- **Symptom:** "This means that..." / "In other words..." / "What this is really saying is..."
- **Why it fails:** You're adding commentary, not extracting
- **Fix:** Delete all explaining. State principle directly.

**FAILURE #2: Context Padding**
- **Symptom:** Adding background, history, or "why this matters"
- **Why it fails:** Makes output longer than input (expansion not compression)
- **Fix:** Context goes in Integration Map, not in distilled principle

**FAILURE #3: Clever Over Clear**
- **Symptom:** Using big words or abstract concepts to sound sophisticated
- **Why it fails:** Loses clarity, needs explanation
- **Fix:** 5-year-old test - could a child grasp the core idea?

**FAILURE #4: Too Much Compression**
- **Symptom:** Extract so short it becomes cryptic
- **Why it fails:** Violates clarity check (needs explanation to understand)
- **Fix:** Add back words until clear, even if slightly longer

**FAILURE #5: Missing the Core**
- **Symptom:** Extracting interesting but peripheral detail instead of central pattern
- **Why it fails:** Not the actual gold
- **Fix:** Re-scan with "what's the ONE core thing here?" question
