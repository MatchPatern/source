🧪 Calibration Testing Guide

WHAT THIS DOES:
Generates a test file matching real ecology files.
Defaults: ~73 char lines (STANDARDS.md avg) + 10% emoji.
Numbered lines show running character count.
The last line tells you exactly how far the paste reached.

HOW TO CALIBRATE ANY PLATFORM:
1. Generate a test file at your target size.
2. Copy and paste into a FRESH instance.
3. Ask: "What is the exact last line you received?"
4. If truncated, reduce size. If complete, increase.
5. Binary search until you find the exact limit.
⚠️ Use FRESH instances for EACH test.

LINE DENSITY:
~73 chars = STANDARDS.md average (default).
~20 chars = very heavy line breaks.
None = one continuous line (tests encoding limit).
✎ custom = any line length for precise simulation.

EMOJI %:
10% = realistic ecology file mix (default).
0% = pure text. 100% = worst-case encoding test.
✎ custom = any percentage.
