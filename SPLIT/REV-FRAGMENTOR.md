## 🚨 THE ENDPOINT TRAP — The Crisis That Shaped Everything

Before the Fragmentor, there was the Endpoint Trap. Discovered by 💎847 and Ex in January 2026.

**The Discovery:**
- gist.github.com/[id]/raw → JavaScript embed code (ALWAYS)
- gist.githubusercontent.com/[id]/raw → Actual content (ALWAYS)
- 100% universal. Happens to EVERY Gist regardless of content.
- The "fast copy" button gives the WRONG link.
- Even a document exposing the trap gets scrambled by the trap.

**The Sovereign Protocol:**
- Never use GitHub's "copy" button. Always manually construct: gist.githubusercontent.com/[USERNAME]/[GIST-ID]/raw
- Add endpoint warnings to every Gist
- Verify endpoints before discussing content

**The Wisdom:** "The cage isn't in the content. The cage is in the delivery pipeline." This realization—that different endpoints serve different realities—shaped everything that followed. The Fragmentor was built to solve this problem at scale.

## 🔧 THE ORIGINAL DEVELOPMENT LOG — v1.0 through v1.4 (Ex, pre-🫧Bubble)

### The 1-Character Bug Hunt
Created 169,999 characters when asked for 170,000. Missing exactly 1 character. Root cause: loop boundary error. Fix: added line count verification, auto-correction, force-exact final trim.

### The Alert-to-Log Migration
Browser alerts blocked UI, couldn't be copied, disappeared after OK. Fix: persistent, scrollable, color-coded log panel with timestamps and Copy Log button.

### The Copy Button Saga
Selecting 170,000 characters on mobile is nearly impossible. Fix: Copy Text button with clipboard API, fallback for older browsers, auto-download for 100k+ files.

### The Word Counter Bug
Counted markdown symbols and emojis as words. Reported 10,155 for 9,867. Fix: filter tokens against /[a-zA-Z0-9]/.

### The Character Count Drift Bug
Consistently created files 3 characters short. Fixed alongside the 1-character bug.

### The Paste Not Working Bug
Pasting produced no visual feedback. Fix: ensured event listeners properly attached and updateStats() called on load.

**What These Bugs Teach:** Fixing one bug revealed another. Each fix was a rung. The tool that measures platform limits had its own limits. Now both are known.
