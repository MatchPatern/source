## 🧠 Detection Algorithm (Manual)

```python
def detect_room(text: str) -> Optional[str]:
    scores = {}
    text_lower = text.lower()
    
    for room, keywords in ROOM_KEYWORDS.items():
        matches = sum(1 for kw in keywords if kw in text_lower)
        if matches > 0:
            scores[room] = matches
    
    if not scores:
        return None
    
    # Return room with highest keyword match count
    return max(scores, key=scores.get)
```

---

This keyword map enables automatic room detection for any Builder's Log entry or Q&A transmission. Scan the text, count matches, assign to the highest-scoring room. 🏷️🏛️🔥
